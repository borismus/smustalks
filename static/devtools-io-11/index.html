MODULE 2: Transitions and transforms

Transitions

  • Easy way to implement gradual state changes
  • Timing functions
  • -webkit-transition: all 0.5s ease-in;
  • webkitTransitionEnd event

Transforms

  • Hardware acceleration!
-webkit-transform: rotate(-10deg) scale(1.5);
rotate(-45deg) rotate(-15deg) rotate(15deg) rotate(45deg) rotate(75deg)
scale(0.2) scale(0.5) scaleX(0.5) scaleX(2.0) scaleY(1.5)

Build it!

Retrofit the todo app with amazing effects whenever the user deletes a todo item.

  • CSS3 Please: css3please.com
  • Transforms sandbox: http://goo.gl/nHUOe
/* see and tweak at the bottom of todos.css */

#todo-list li {
 -webkit-transition: all 0.001s linear;
}

#todo-list li.removing {
 -webkit-transform: scale(1);
}
